# 사주 계산 엔진 설계 문서

**날짜:** 2026-06-09  
**대상 파일:** `lib/saju-data.ts`, `lib/saju-calculator.ts`

---

## 개요

생년월일시를 받아 사주팔자(4기둥 8글자)와 오행 분포를 계산하는 엔진.  
절기 기반 월주, 지장간 포함 오행 분포를 지원한다.

---

## 파일 구조

```
lib/
├── saju-data.ts        # 정적 데이터 테이블 (천간·지지·지장간·절기)
└── saju-calculator.ts  # 계산 로직 + 공개 API
```

## 타입 정의

```typescript
type Ohaeng = '목' | '화' | '토' | '금' | '수';

interface Pillar {
  gan: string;          // 천간 한자
  ji: string;           // 지지 한자
  ganElement: Ohaeng;
  jiElement: Ohaeng;
}

interface SajuResult {
  year: Pillar;
  month: Pillar;
  day: Pillar;
  hour: Pillar | null;  // 시간 모름이면 null
  ilgan: string;        // 일간 (운세·궁합 기준)
  ohaeng: Record<Ohaeng, number>;
}

interface SajuInput {
  year: number;
  month: number;        // 1-12, 양력 기준
  day: number;
  hour: number | null;  // null = 모름, 0-23
  isLunar: boolean;
}
```

## saju-data.ts — 데이터 테이블

### 천간 (10개)
甲乙丙丁戊己庚辛壬癸 — 각각 오행(목·화·토·금·수)과 음양 매핑

### 지지 (12개)
子丑寅卯辰巳午未申酉戌亥 — 각각 오행과 음양 매핑

### 지장간 테이블
지지별로 [여기(餘氣), 중기(中氣), 정기(正氣)] 천간을 보유.  
중기가 없는 지지(子·午·卯·酉)는 배열 길이 2.

| 지지 | 여기 | 중기 | 정기 |
|------|------|------|------|
| 子   | 壬   | —    | 癸   |
| 丑   | 癸   | 辛   | 己   |
| 寅   | 戊   | 丙   | 甲   |
| 卯   | 甲   | —    | 乙   |
| 辰   | 乙   | 癸   | 戊   |
| 巳   | 戊   | 庚   | 丙   |
| 午   | 丙   | —    | 丁   |
| 未   | 丁   | 乙   | 己   |
| 申   | 戊   | 壬   | 庚   |
| 酉   | 庚   | —    | 辛   |
| 戌   | 辛   | 丁   | 戊   |
| 亥   | 戊   | 甲   | 壬   |

### 12절기(節) 목록
소한·입춘·경칩·청명·입하·망종·소서·입추·백로·한로·입동·대설  
(각 월의 시작을 결정. 중기는 월주 계산에 사용하지 않음)

## saju-calculator.ts — 계산 로직

### 음력 변환
`isLunar === true`이면 `lunar-javascript`의 `Lunar.fromYmd(y, m, d).getSolar()`로 양력 변환 후 처리.

### 년주(年柱)
1. 해당 년도의 입춘(立春) 날짜를 `SolarTerm`으로 조회
2. 입력일이 입춘 이전이면 `targetYear = year - 1`
3. 60갑자 인덱스 = `(targetYear - 4) % 60` (음수 방지: `((n % 60) + 60) % 60`)
4. 인덱스로 천간·지지 결정

### 월주(月柱)
1. 12절기(節) 날짜를 해당 년도 앞뒤로 조회
2. 입력일 직전 절기를 찾아 월지(月支) 인덱스 결정 (인월=0, 묘월=1, …)
3. 월간(月干)은 **오호둔법**으로 결정:

| 년간 | 인월 시작 月干 |
|------|--------------|
| 甲·己 | 丙 |
| 乙·庚 | 戊 |
| 丙·辛 | 庚 |
| 丁·壬 | 壬 |
| 戊·癸 | 甲 |

### 일주(日柱)
- 기준일: 1900-01-31 = 甲戌 (60갑자 인덱스 10)
- 인덱스 = `(daysDiff + 10) % 60`
- `daysDiff`: 기준일부터 입력일까지의 일수 (UTC 기준 자정)

### 시주(時柱)
- 시간이 `null`이면 시주 = `null`
- 12시진 매핑 (자시는 23:00~00:59):

| 시진 | 지지 | 시간 범위 |
|------|------|----------|
| 자시 | 子   | 23·0     |
| 축시 | 丑   | 1·2      |
| 인시 | 寅   | 3·4      |
| 묘시 | 卯   | 5·6      |
| 진시 | 辰   | 7·8      |
| 사시 | 巳   | 9·10     |
| 오시 | 午   | 11·12    |
| 미시 | 未   | 13·14    |
| 신시 | 申   | 15·16    |
| 유시 | 酉   | 17·18    |
| 술시 | 戌   | 19·20    |
| 해시 | 亥   | 21·22    |

- 시간(時干)은 **오자둔법**으로 결정:

| 일간 | 자시 시작 時干 |
|------|--------------|
| 甲·己 | 甲 |
| 乙·庚 | 丙 |
| 丙·辛 | 戊 |
| 丁·壬 | 庚 |
| 戊·癸 | 壬 |

### 오행 분포
8글자 + 지장간을 순회하며 오행 점수 누적:

| 항목 | 점수 |
|------|------|
| 천간 오행 (4개) | 각 1.0점 |
| 지지 표면 오행 (4개) | 각 1.0점 |
| 지장간 여기 (4개) | 각 0.5점 |
| 지장간 중기 (중기가 있는 지지만: 丑寅辰巳未申戌亥) | 각 0.5점 |
| 지장간 정기 (4개) | 각 1.0점 |

## 공개 API

```typescript
export function calculateSaju(input: SajuInput): SajuResult
```

단일 진입점. 내부적으로 년주→월주→일주→시주 순으로 계산하고 오행 분포를 집계해 반환.

---

## 제약 사항

- 지원 범위: 1900년~2100년 (lunar-javascript 지원 범위)
- 시간대: 한국 표준시(UTC+9) 기준 입력을 가정. 별도 시간대 변환 없음
- 야자시(夜子時, 23:00~23:59)는 당일 자시로 처리 (일주는 다음날로 넘어가지 않음)
